## Overview

A conversation bewteen three participants that can lead to the identification of a bird based on the description given in the user prompt.
Tools such as retrieving yjr time, news, weather, or bird sightings are available for the participants to use 

## Set up

You will also need to include the api keys from OpenAI and eBird in your .env file before running this project.

```sh
uv sync
uv run python main.py
```
